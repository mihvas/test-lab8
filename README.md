# test-lab8

# Selenium OpenFoodFacts

## Установка

```bash
pip install -r requirements.txt
```

Запуск tests

```bash
pytest
```

Генерация отчета Allure

```bash
pytest --alluredir=allure-results
```

Открыть отчёт

```bash
allure serve allure-results
```

## Technologies

- Python
- Selenium
- pytest
- Allure
- WebDriverManager
- Page Object Model
